## 멤버 추가

회고에 참여하는 멤버가 추가된 경우 프로젝트 최상단의 [members.json](./members.json)에 자신의 깃허브 계정과 이름을 추가해주세요.
깃허브 계정이 `aA-zZ` 순서로 올 수 있게 정렬해주세요. (대문자는 소문자보다 아래에 있어야 합니다.)

```js
{
    ...
    "[GitHub Account]": "[Your Name]",
    ...
}
```

## 이슈 등록 규칙

- 플랫폼으로 전송되는 링크로 이슈를 생성하면 오늘 날짜가 자동으로 삽입됩니다.
- 이슈를 직접 등록하는 경우 제목의 YYYY-MM-DD 부분은 오늘 날짜로 해당 양식에 맞게 수정해주세요.
- 그 외에는 수정할 필요 없이 본문만 작성하여 등록해주면 됩니다.
  - 제목의 XXX는 깃헙 액션을 통해 자동으로 작성자의 이름으로 변경됩니다.
  - 할당자도 깃헙 액션을 통해 자동으로 작성자가 추가됩니다.
